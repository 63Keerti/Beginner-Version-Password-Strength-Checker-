# 🔐 Password Strength Checker

A simple and interactive web-based Password Strength Checker that helps users create secure passwords by analyzing their strength in real-time.

---

## 🚀 Features

- Password strength meter (Weak / Medium / Strong)
- Show / Hide password toggle
- Password generator
- Real-time validation
- Common password detection
- Security tips display
- Crack time estimation
- Animated progress bar
- Responsive UI design
- Dark mode support

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

---

## 🎯 Project Goal

To help users understand password security basics and improve their password strength using real-time feedback.

---

## 📸 Output

- Strength meter bar
- Security suggestions
- Generated strong passwords

---

## 📚 Learning Outcomes

- DOM manipulation
- Form validation
- Basic cybersecurity concepts
- UI/UX design basics
